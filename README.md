# Particle Sphere

A premium, interactive particle sphere visualization with gesture control support.

## Features

- Interactive 3D particle sphere
- Real-time particle manipulation
- Webcam gesture control via MediaPipe
- Premium dark aesthetic
- Minimalist design

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 19
- Vite
- MediaPipe for gesture tracking
- Lucide React for icons
